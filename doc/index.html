About

ebiflyはiPhoneでのWeb開発、特にデフォルトのデバッグ機能を利用することができない UIWebViewを用いた開発をサポートすることをコンセプトに作られたJavaScript?デバッグツールです。
alert('hoge')というようなコードを書いてオブジェクトの内容を確認したり、 デバッグ専用コードをJavaScript内に埋め込む必要がないよう、 別のブラウザを母艦としてログの送信、JavaScriptの実行を行います。
ebiflyは専用のサーバアプリケーションをローカル環境などで実行することで、 デバッグを開始することができます。

Logger

以下のような簡単なコマンドでクライアントからコンソールへログを送信することができます。

Script

コンソールから送信されたスクリプトはiPhone上で実行され、その結果をコンソールに返します。

Node.js + WebSocket

ebiflyはnode.js、およびsocket.ioを利用して通信を行うので、こちらのインストールが別途必要となります。 socket.ioによって利用可能なブラウザではWebSocketを使用し、さらにサーバサイドをJavaScriptで実装しているため、 JavaScriptのみですべてのスクリプトを変更することができます。

Extension
